# Cleaning Spotify Unpopular Songs (2022)

This project documents the end-to-end cleaning of a Spotify songs dataset restricted to tracks with a **popularity score below 10/100**. The goal was to transform raw Spotify metadata into a clean, structured dataset suitable for analysis.

## Dataset Scope
- Spotify tracks released in 2022
- Songs filtered to low-popularity entries (popularity < 10)

## Key Cleaning Steps
- Filtered dataset to retain only unpopular tracks
- Inspected and handled missing and inconsistent values
- Standardized column formats and data types
- Removed duplicates and irrelevant attributes

## Tools & Environment
- R
- Tidyverse package

## What This Project Demonstrates
- Data cleaning and preprocessing skills  
- Ability to work with real-world, imperfect datasets  

## Output
A cleaned Spotify dataset ready for exploratory data analysis or machine learning workflows.
